## Overview of the content of the folder `01-resources`

This folder contains files that are unchangeable resources for the analyses of this project.

- `ancient_dental_calculus_samples.tsv`: the list of all ancient dental calculus samples analysed in this study
- `ancient_dental_calculus_merging.tsv`: the list of which ancient dental calculus samples and their respective sequencing runs were merged for assembly
- `AncientMetagenomeDir_v23.06.0_dentalcalculus_samples.tsv`: overview of the dental calculus samples present in the AncientMetagenomeDir database in June 2023
- `ENA_contigs_analysis_codes.tsv`: the overview of the ENA analysis and submission codes when uploading the _de novo_ assembled contigs
- `modern_dental_calculus_samples.tsv`: the overview of the modern dental calculus samples _de novo_ assembled in this study
- `modern_saliva_samples.tsv`: the overview of the modern saliva samples _de novo_ assembled in this study
- `modern_supragingivalplaque_samples.tsv`: the overview of the modern supragingival plaque samples _de novo_ assembled in this study
- `WarinnerLab_ancient_dentalcalculus_samples.tsv`: the overview of (un)published ancient dental calculus samples from the Warinner lab after June 2023
