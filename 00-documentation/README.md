## Overview of the folder `00-documentation`

This folder contains documentation and metadata tables for samples and MAGs in this project, including summary tables produced by the analysis workflows. These files are used for generating figures from scripts in 06-publication/main_figures`

- `00-documentation/ERSX_to_ERS_conversion_table.tsv` - Sample label conversion table
- `00-documentation/ancient_envt_bins_26_07_2024.tsv` - bins considered to be environmental contamination
- `00-documentation/ancient_envt_clusters_26_07_2024.tsv` - species clusters considered to be environmental contamination
- `00-documentation/class_gram_stains.tsv` - Gram stain of microbial taxonomic Classes
- `00-documentation/cluster_taxonomy_for_taxdump_26_07_2024.tsv` - taxonomy input table for generating Kraken2 database
- `00-documentation/cluster_type_table.tsv` - Host information per species cluster
- `00-documentation/cmc_accession_codes_published.tsv` - Sample label conversion table
- `00-documentation/continent_coordinates.tsv` - coordinates for continents for plotting
- `00-documentation/csc001_taxonomy.tsv` - taxonomy for sample CSC001
- `00-documentation/csc_mags_info.tsv` - metadata for sample CSC001
- `00-documentation/ersx_code_conversion_table.tsv` - Sample label conversion table
- `00-documentation/lib_metadata_general.tsv` - sample metadata
- `00-documentation/lib_metadata_pca.tsv` - sample metadata
- `00-documentation/lib_metadata_sp_curves.tsv` - sample metadata
- `00-documentation/mag_quality_table.tsv` - MAG quality estimates from checkM, checkM2, GUNC, etc
- `00-documentation/mag_sample_to_eager_label.tsv` - Sample label conversion table
- `00-documentation/mod_anc_refined_mag_taxonomy_26_07_2024.tsv` - taxonomy for all MAGs that passed quality thresholds
- `00-documentation/poor_samples.tsv` - poorly preserved samples that were removed from analyses that used metagenomic taxonomic profiling
- `00-documentation/qual_fail_or_non_oral_list_mag_source.tsv` - all MAGs that didn't pass quality checks
- `00-documentation/rgig7111_anaerovoracaceae_complete_metadata_09_2025.tsv` - metadata for for Anaerovoracaceae genomes
- `00-documentation/zhu2022_accessions.csv` - sample info for Zhu, et al. 2022 MAGs
